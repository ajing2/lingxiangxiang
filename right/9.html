frame标签

代码：
<!doctype html>
<html lang="en">
<head>
    <title>IT-xiang</title>
</head>
<frameset rows = "20%, *">
    <frame src = "top.html"></frame>
    <frameset cols = "15%, *">
    <frame src = "left.html"></frame>
    <frame src = "right.html" name = "content"></frame>
</frameset>
</html>
效果：
https://ajing2.github.io/lingxiangxiang/index.html
说明：

1.frame标签写的网页不能被百度等收录，认为这个是不安全的
2.rows = "20%, 30%, *" 分成2，3, 5 三份，*表示剩下的所有
3.<a href = "right/1.html" alt = "目录" target = "content">html简介</a>目录进行连接时，要设置target属性。
需要和    <frame src = "right.html" name = "content"></frame>中的name保持一致
4.frame不能和body等body中的标签同时存在，否则不显示
