css的基本规则

css基本规则：选择器+一个或者多个声明


1.html选择器：html元素+{}
<head>
<style type = "text/css">
h4{font-size:3px;color:red;}
</style>
</head>
<h4>html选择器</h4>

效果：

html选择器


2.id选择器：#+id_name+{}
<head>
<style type = "text/css">
#xiang{color:red;font-size:30px;}
</style>
</head>
<h4>this is #xiang id</h4>

效果：

this is #xiang id


3.类选择器：.+class_name+{}
<head>
<style type = "text/css">
.xiang{color:blue;font-size:30px;}
</style>
</head>
<h4>this is #xiang class</h4>

效果：

this is #xiang class
