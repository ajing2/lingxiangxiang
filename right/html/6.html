无序列表

格式：
<ul type="disc/circle/square">
<li>列表项1</li>
<li>列表项2</li>
</ul>
type = "disc"：典型的小黑圆圈
type = "circle"：空心小圈
type = "square"：黑方块

效果：
  • 列表项1
  • 列表项2

有序列表

格式：
<ol type = "1/A/a/I/i">
<li>有序列表项1</li>
<li>有序列表项2</li>
</ol>
type = "1" 默认是数字
type = "A" 按照A-Z排序
type = "a" 按照a-z排序
type = "I" 阿拉伯I排序
type = "i" 阿拉伯i排序

效果：
  1. 有序列表项1
  2. 有序列表项2

自定义列表

格式：
<dl>
<dt>自定义列表1</dt>
<dd>111111</dd>
<dd>222222</dd>
<dd>333333</dd>
<dt>自定义列表2</dt>
<dt>自定义列表3</dt>
</dl>

效果：
自定义列表1
111111
222222
333333
自定义列表2
自定义列表3
